## 远程操作服务器


接口文档查看 \
    http://127.0.0.1:8000/docs \
    http://127.0.0.1:8000/redoc \
可选择是否开启接口文档
在main.py中找到docs_url=None，redoc_url=None进行控制
---
第一次执行时需要创建数据库
在main.py中
```python
# 初始化数据库 只需要执行一次
from utils import sqlite_db
sqlite_db.ConnDb().init_table()
```
